What You Need

  1. Azure OpenAI Service:
    • Endpoint URL (e.g., https://your-resource.openai.azure.com/)
    • API Key
    • Deployment name (e.g., gpt-4o)
  2. Azure Speech Services:
    • Subscription Key
    • Region (e.g., eastus)

Setup Instructions

Replace the placeholder values in the code below with your actual Azure credentials:

LiaPlusVoice.init({
  mode: 'custom',
  azureOpenAI: {
    endpoint: 'https://YOUR-RESOURCE.openai.azure.com/',
    apiKey: 'YOUR_OPENAI_API_KEY',
    deployment: 'gpt-4o'
  },
  azureSpeech: {
    key: 'YOUR_SPEECH_API_KEY',
    region: 'eastus'
  },
  language: 'en-US'
});

Use Cases for Custom Mode

  • Development/Testing: Test with your own Azure resources
  • Enterprise: Use your organization's Azure subscriptions
  • Custom Regions: Deploy in specific Azure regions for compliance
  • Cost Control: Monitor and control your own Azure usage
